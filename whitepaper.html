Whitepaper

1. Abstract

FortuneFI is a MemeFi Loot Game that combines NFTs, DeFi, and meme culture into a fair, transparent ecosystem. Instead of jackpots, users earn NFT Tickets that grant access to Loot Pools funded by every $FFI transaction.

This creates a system where anyone can play for free, collect tradable NFTs, and access community reward events — legally framed as GameFi, not gambling.

2. Problem Statement

  • Traditional lotteries = centralization, licenses, barriers.
  • Classic P2E = grindy, inflationary.

FortuneFI solves this by:

  • NFT Tickets (collectible + tradable)
  • Loot Pools (transparent & growing with every transaction)
  • MemeFi branding (community-driven virality)

3. Solution

Three participation categories:

  • Free-to-Play: 1 free spin daily — rewards: $FFI, Common NFTs, extra spins.
  • Standard Tickets: Buy NFT Tickets with $FFI (Common/Silver/Gold) — access to Loot Pools + staking boosts.
  • Premium Tickets: Diamond Tickets (only $FFI) — bigger odds, stronger boosts, RWA rewards, VIP Loot Drop events.

4. Loot Pools

Every $FFI transaction funds the community treasury.

  • 55% Monthly
  • 20% Quarterly
  • 25% Yearly

Rewards: $FFI airdrops, NFT upgrades, RWA fractions. Framed as Loot Drops, not jackpots.

7. Presale

Whitelist → Presale rounds (Seed/Private/Public).

  • Seed = Gold Ticket
  • Private = Silver Ticket
  • Public = Common Ticket

Presale tx-ids = Founders Wheel entries (extra NFT rewards).

8. Roadmap

  1. Phase 1: Presale, NFT Tickets mint, Loot Pool contract launch.
  2. Phase 2: Daily spin + staking with NFT boosts.
  3. Phase 3: RWA-backed Loot Drops.
  4. Phase 4: CEX listings + MemeFi viral expansion.

9. Community

  • Leaderboards: Lucky winners & meme lords.
  • Referral rewards: more spins & NFT tickets.
  • Meme contests with NFT prizes.

10. Conclusion

FortuneFI is not a lottery. It’s a community Loot Game powered by $FFI, NFT Tickets, and meme culture. Every on-chain transaction fuels growth, every NFT adds value, and every player has a chance to Dream Big & Win Bigger.
